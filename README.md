# 3D Palmprint Biometric Recognition

## Overview
This project implements a biometric recognition system using pseudo-3D reconstruction and deep learning.

## Methodology
- 3D Reconstruction using Frankot–Chellappa
- Feature extraction using HSANet, MobileNetV3
- Classification using supervised learning

## Results
- Achieved near 100% accuracy on IIT Delhi dataset
- Evaluated using ROC, CMC, and EER

## Deployment
- Implemented on NVIDIA Jetson Nano for real-time inference

## Tech Stack
Python, MATLAB, Deep Learning
